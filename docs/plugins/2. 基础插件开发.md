# 2. 基础插件开发

## 文档概述

本文档将带你从零开始创建一个简单的 Vite 插件，通过实际案例学习插件开发的基本流程和技巧。

**学习目标：**

- 掌握 Vite 插件的基本结构
- 学会创建简单的 Vite 插件
- 理解插件钩子的使用方法
- 能够处理自定义文件类型

## 一、插件开发准备

### 1.1 环境准备

在开始插件开发前，确保你已经具备以下环境：

- Node.js 16+ 环境
- Vite 项目（可以是新创建的项目）
- 基本的 TypeScript 知识

### 1.2 项目结构

创建一个基础的插件开发项目结构：

```
my-vite-plugin/
├── package.json
├── src/
│   └── index.ts      # 插件主文件
├── tsconfig.json
└── vite.config.ts    # 测试配置
```

## 二、示例一：给代码添加版权注释

### 2.1 插件功能说明

创建一个插件，自动给 JavaScript 和 TypeScript 文件添加版权注释。

### 2.2 插件实现

**创建插件文件** **`src/index.ts`：**

```typescript
export function copyrightPlugin(options?: {
  author?: string;
  year?: number;
  license?: string;
}) {
  const { author = 'Unknown', year = new Date().getFullYear(), license = 'All rights reserved' } = options || {};
  
  const copyrightNotice = `/**
 * Copyright © ${year} ${author}
 * ${license}
 */\n`;
  
  return {
    name: 'vite-plugin-copyright',
    
    transform(code: string, id: string) {
      // 只处理 JavaScript 和 TypeScript 文件
      if (id.match(/\.(js|ts|jsx|tsx)$/)) {
        // 检查是否已经有版权注释
        if (!code.trim().startsWith('/**')) {
          return {
            code: copyrightNotice + code,
            map: null
          };
        }
      }
      return null;
    }
  };
}

export default copyrightPlugin;
```

### 2.3 插件使用

**在** **`vite.config.ts`** **中使用：**

```typescript
import { defineConfig } from 'vite';
import react from '@vitejs/plugin-react';
import { copyrightPlugin } from './src';

export default defineConfig({
  plugins: [
    react(),
    copyrightPlugin({
      author: 'My Company',
      year: 2024,
      license: 'MIT License'
    })
  ]
});
```

### 2.4 测试验证

创建一个测试文件 `src/test.ts`：

```typescript
export function hello() {
  return 'Hello Vite!';
}
```

运行 Vite 构建，查看生成的文件是否自动添加了版权注释。

## 三、示例二：处理自定义文件后缀

### 3.1 插件功能说明

创建一个插件，处理 `.yaml` 和 `.yml` 文件，将其转换为 JavaScript 对象。

### 3.2 依赖安装

首先安装 YAML 解析库：

```bash
npm install yaml
npm install -D @types/yaml
```

### 3.3 插件实现

**更新** **`src/index.ts`：**

```typescript
import yaml from 'yaml';

export function yamlPlugin() {
  return {
    name: 'vite-plugin-yaml',
    
    // 解析 YAML 文件
    resolveId(source: string) {
      if (source.endsWith('.yaml') || source.endsWith('.yml')) {
        return source;
      }
      return null;
    },
    
    load(id: string) {
      if (id.endsWith('.yaml') || id.endsWith('.yml')) {
        try {
          // 读取 YAML 文件内容
          const fs = require('fs');
          const content = fs.readFileSync(id, 'utf-8');
          // 解析 YAML
          const data = yaml.parse(content);
          // 转换为 JavaScript 模块
          return `export default ${JSON.stringify(data, null, 2)}`;
        } catch (error) {
          console.error('Error parsing YAML:', error);
          return null;
        }
      }
      return null;
    }
  };
}

export function copyrightPlugin(options?: {
  author?: string;
  year?: number;
  license?: string;
}) {
  // 保持之前的实现...
  const { author = 'Unknown', year = new Date().getFullYear(), license = 'All rights reserved' } = options || {};
  
  const copyrightNotice = `/**
 * Copyright © ${year} ${author}
 * ${license}
 */\n`;
  
  return {
    name: 'vite-plugin-copyright',
    
    transform(code: string, id: string) {
      // 只处理 JavaScript 和 TypeScript 文件
      if (id.match(/\.(js|ts|jsx|tsx)$/)) {
        // 检查是否已经有版权注释
        if (!code.trim().startsWith('/**')) {
          return {
            code: copyrightNotice + code,
            map: null
          };
        }
      }
      return null;
    }
  };
}

export default copyrightPlugin;
```

### 3.4 插件使用

**在** **`vite.config.ts`** **中使用：**

```typescript
import { defineConfig } from 'vite';
import react from '@vitejs/plugin-react';
import { yamlPlugin, copyrightPlugin } from './src';

export default defineConfig({
  plugins: [
    react(),
    yamlPlugin(),
    copyrightPlugin()
  ]
});
```

### 3.5 创建测试文件

**创建** **`src/config.yaml`：**

```yaml
app:
  name: 'My App'
  version: '1.0.0'
  description: 'A Vite app with YAML support'
  
server:
  port: 3000
  host: 'localhost'
  
database:
  url: 'mongodb://localhost:27017'
  name: 'myapp'
```

**在** **`src/App.tsx`** **中使用：**

```typescript
import React from 'react';
import config from './config.yaml';

function App() {
  return (
    <div>
      <h1>{config.app.name}</h1>
      <p>Version: {config.app.version}</p>
      <p>Description: {config.app.description}</p>
      <p>Server: {config.server.host}:{config.server.port}</p>
    </div>
  );
}

export default App;
```

## 五、发布插件

### 5.1 准备发布

1. **更新 package.json**：
   - 添加 `name`、`version`、`description` 等字段
   - 添加 `main`、`module`、`types` 字段
   - 添加 `keywords` 字段，包含 `vite-plugin`
2. **创建 README.md**：
   - 插件功能说明
   - 安装方法
   - 使用示例
   - 配置选项
3. **构建插件**：
   ```bash
   npm run build
   ```

### 5.2 发布到 npm

1. **登录 npm**：
   ```bash
   npm login
   ```
2. **发布插件**：
   ```bash
   npm publish
   ```

## 篇章概述

插件篇将带你深入学习 Vite 插件的开发，从入门到高级应用。

## 学习内容

### 1. 插件入门
- 插件的作用
- 常用插件的安装与使用
- Vite 插件的基本结构与钩子机制
- 自定义插件示例
- 第三方插件集成

### 2. 基础插件开发
- 创建一个简单的 Vite 插件
  - 给代码添加版权注释
  - 处理自定义文件后缀

### 3. 高级插件开发
- 开发复杂 Vite 插件
  - 自定义打包逻辑
  - 集成第三方工具
  - 实现自定义热更新
- 插件调试方法
  - 使用 debug 工具调试插件代码
- 实操：开发一个用于业务场景的插件
  - 自动生成接口请求文件
  - 处理后端返回的 Mock 数据

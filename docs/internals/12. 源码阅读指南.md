## 12.3 关键流程

### 12.3.1 开发服务器启动流程

```mermaid
flowchart TD
    A[执行 vite] --> B[解析配置]
    B --> C[依赖预构建]
    C --> D[启动 HTTP 服务器]
    D --> E[加载插件]
    E --> F[启动监听]
    F --> G[等待请求]
```

### 12.3.2 请求处理流程

```mermaid
flowchart TD
    A[浏览器请求] --> B[中间件处理]
    B --> C[resolveId]
    C --> D[load]
    D --> E[transform]
    E --> F[返回代码]
```

## 12.8 贡献指南

### 12.8.1 提交代码流程

```bash
# 1. 从 main 创建分支
git checkout main
git checkout -b my-feature

# 2. 编写代码

# 3. 运行测试
pnpm test

# 4. 提交代码
git add .
git commit -m "feat: add my feature"
```

### 12.8.2 运行测试

```bash
# 运行所有测试
pnpm test

# 运行特定测试
pnpm test vitest/basic

# 运行单个测试文件
pnpm test packages/vite/__tests__/build.ts
```

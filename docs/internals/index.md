# 📚 Vite 原理篇

欢迎来到 Vite 原理篇！本篇章将深入探讨 Vite 的核心原理，帮助你理解 Vite 为什么快、如何工作，以及如何阅读和调试 Vite 源码。

---

## 📖 篇章概述

原理篇将从架构层面到具体实现，系统讲解 Vite 的核心工作原理，包括：

- 🔧 **核心架构**：Vite 的整体设计思路
- ⚡ **性能特性**：依赖预构建、缓存机制等
- 🔥 **HMR 原理**：热模块替换的底层实现
- 📦 **构建流程**：开发环境 vs 生产环境
- 🔌 **插件系统**：插件钩子的底层实现
- 📖 **源码阅读**：如何阅读和调试 Vite 源码

## 🎯 学习建议

### 学习顺序

1. **先学核心架构**（第 1-2 章）：建立全局认知
2. **再学核心功能**（第 3-7 章）：理解关键特性
3. **然后学高级特性**（第 8-11 章）：深入细节
4. **最后学源码阅读**（第 12 章）：动手实践

### 学习重点

- ⭐ **依赖预构建**：理解 Vite 性能优势的关键
- ⭐ **缓存机制**：理解为什么 Vite 启动快
- ⭐ **HMR 原理**：理解开发体验好的原因
- ⭐ **整体架构**：建立全局认知

### 学习方法

每一章都包含**原理讲解**和**实验验证**两部分，建议：

1. 先阅读原理讲解
2. 然后亲手做实验验证
3. 最后对比实验结果加深理解

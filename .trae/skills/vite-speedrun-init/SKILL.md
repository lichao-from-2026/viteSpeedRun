# Vite Speed Run 初始化技能

## 功能说明

此技能用于在每次对话开始时读取项目的关键配置文件，确保了解：
1. 项目当前状态和学习进度
2. 项目约束和操作规范

## 使用时机

**⚠️ 强制要求：在任何对话或执行任何操作之前，必须先调用此技能！**

特别是以下情况：
- 新对话开始时
- 上下文可能被压缩后
- 不确定项目当前状态时

## 操作步骤

1. 读取 `.trae/CONTEXT.md` - 获取项目上下文
2. 读取 `.trae/CONSTRAINTS.md` - 获取约束规范
3. 理解项目当前状态和学习进度
4. 根据用户需求执行操作

## 关键要求

### 绝对不能忘记的事情：

1. **每次对话前必须先调用此技能**
2. **生成的所有代码和配置必须有详细中文注释**
3. **不要主动创建文档类文件，除非用户明确要求**
4. **优先编辑现有文件，不轻易创建新文件**
5. **不要污染 src/ 目录的生产代码**

### 项目上下文关键信息

- 项目名称：Vite 学习实验项目
- 技术栈：React 19 + TypeScript 6 + Vite 8
- 学习路径记录在 `.trae/CONTEXT.md` 中
